## Local installation
git clone https://github.com/iee-ihu-gr-course1941/ADISE20_score_4.git

from the backend directory:
```
$ npm install
$ npm start
```
If there are any issues, you may need to upgrade your node version.

from the frontend directory:
```
$ npm install
$ npm start
```
